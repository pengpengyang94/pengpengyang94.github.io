Mafft快速入门（Windows版）

April 17, 2020 in 软件说明

　　一提到多重序列比对，很多人禁不住就想到ClustalW（Clustalx为ClustalW的GUI版），其实有一款多重序列比对软件－MAFFT，不论从比对速度（Muscle>MAFFT>ClustalW>T-Coffee），还是比对准确性（MAFFT>Muscle>T-Coffee>ClustalW）来说，其相比于ClustalW（或ClustalX）有过之而无不及，所以这里强烈推荐使用MAFFT这款多重比对软件。
　　

1.下载

　　从mafft官网下载所需版本的软件，下图展示的是本文所需的的 Windows 版mafft软件。

注意：

  • mafft是免安装的软件，解压即用！！！
  • 解压后的存放路径必须全英文路径，否则不能正常打开！！！

2.使用

1.cmd 切换至 mafft 解压后的路径，推荐使用 cmdhere 进入
2.打开 mafft，官方提供两种方法进入 mafft 界面：

  • 双击 mafft-win文件夹中的 mafft.bat
  • 右键 mafft-signed.ps1 后，点击 使用Power Shell 运行

mafft主界面如下：

3.按照提示输入input文件的路径
　　本文演示的 mafft 是直接解压在桌面，同时所需比对的序列文件是1.fa，输入以下指令即可。

# 建议使用绝对路径，具体看个人偏好
C:\Users\yangp\Desktop\mafft-win\1.fas

4.按照提示输入output文件相关信息
　　要输入 output 文件名称和 output 文件存放位置。
　　本文将输出文件称为1.fa，并置于 mafft-win文件夹中，故输入下面指令。

# 建议使用绝对路径，具体看个人偏好
C:\Users\yangp\Desktop\mafft-win\1.fas

5.选择输出的格式
　　有6种格式可供选择，根据自己的要求进行选择，一般选择3或4。（注意：如果下一步采用fasttree进行建树的话，必须选择3或4，其他格式fasttree不支持）

6.选择比对策略
　　根据需求选择，不知道需求选择 1. --auto 即可

7.是否需要添加其他选项
　　没有需求直接回车即可。其他选项信息详见第三部分mafft的详细指令描述。

8.确认以上输入信息
　　输入Y确认信息。

9.进行比对
　　等待比对完成，比对完成后会显示比对结果，空格键为翻页，回车键为下一行，翻到底部后会提示按回车键退出。

3.mafft详细指令描述

# 输入如下指令
mafft.bat -help
# 出现 mafft 的帮助文档
High speed:
  % mafft in > out
  % mafft --retree 1 in > out (fast)

High accuracy (for <~200 sequences x <~2,000 aa/nt):
  % mafft --maxiterate 1000 --localpair  in > out (% linsi in > out is also ok)
  % mafft --maxiterate 1000 --genafpair  in > out (% einsi in > out)
  % mafft --maxiterate 1000 --globalpair in > out (% ginsi in > out)

If unsure which option to use:

  % mafft --auto in > out

--op # Gap opening penalty, default: 1.53
--ep # Offset (works like gap extension penalty), default: 0.0
--maxiterate # Maximum number of iterative refinement, default: 0
--clustalout :Output: clustal format, default: fasta
--reorder :Outorder: aligned, default: input order
--quiet :Do not report progress
--thread # Number of threads (if unsure, --thread -1)
--dash :Add structural information (Rozewicki et al, submitted)

4.网页版mafft使用指南

MAFFT version 7 Online version

1.打开网页版地址，按照下图进行勾选

2.需要填写正确的邮箱，因为结果发到填写的邮箱中

3.结果邮件的内容如下，点击链接进入结果界面

4.结果界面
　　点击 Fasta format 后，将其全部内容复制到一个新的txt中，保存后将文件类型由txt改为fas（或fasta一类的格式），修改后的文件可直接使用fasttree进行建树

注意：比对结果链接有效期为 96 小时！！！
