码字必备的Markdown语法

May 3, 2020 in 软件说明

写在前面：

  • 本文推荐Typora码字，软件安装见Y01_1_R和Python大礼包（初学必备）.md中Markdown的介绍。
  • 本文主要对码字常用Markdown语法进行说明，如标题、目录、强调、换行、图片与链接、引用、代码、和分割线，其他语法详见文末参考资料。
  • 本文为了便于快速记忆且不混淆，大部语法只对其中最便于使用的方式进行说明。

标题

说明：#后面有空格，一个#代表一级标题，最多6个#代表六级标题。

# 一级标题
###### 六级标题

效果如下：

目录

说明：在所需插入目录的地方写入[TOC]即可，根据个人喜好设置即可，由于大纲能够代替目录的作用，所以为了使文本更加简洁，本文不建议生成目录。

强调

说明：前后加**为加粗；前后加*为倾斜；前后加~~为删除。

**加粗**
*倾斜*
~~删除~~

首行缩进

两种方式：
一是行首添加 &#160; &#160; &#160; &#160; 二是将输入法调成全角输入，然后空格两次即可

换行

说明：文本后加上两个空格表示换行

图片与链接

说明：
　　插入图片与链接的语法只相差一个!，且插入图片语法中()的内容可以为URL或路径，而插入链接中()的内容只能是URL。

注意：
　　上述语法语法只能插入图片，而不能对图片进行调节，所以推荐使用基于html格式的插入图片语法，其中就可以对图片大小及位置进行修改。

引用

说明：使用>+空格+内容

> 引用

代码

说明：

  • 分为行内代码和代码块
  • 行内代码则在其前后加 `
  • 代码块则在其上下行加 ```
  • 代码块上行的 ``` + 空格 + 语言名称可使代码根据语言类型高亮；而在Typora中只需直接输入语言名称即可
  • 在英文输入状态下，使用Tab上方的键即可打出 `

分割线

说明：三个及以上的-代表分割线

脚注

说明：使用[^标记]来定义脚注1。
